# loop
### 程式設計題
```
使用For 迴圈(loop)計算1+2+3+.....100
```
```
sum = 0
for X in range(1,101):
  sum = sum + X
  
print(sum)
```
```
使用For 迴圈(loop)計算1+3+5+7.....+99
```
```
sum = 0
for X in range(1,101,2):
  sum = sum + X
  
print(sum)
```
```
使用for 迴圈(loop)計算1*3*5*7.....*99
```
```
sum = 1
for X in range(1,101,2):
  sum = sum * X
  
print(sum)
```
# while 迴圈(loop)
### 程式設計題

```
使用while 迴圈(loop)計算1+2+3+.....100
```
```
a = 1
sum = 0
while a < 101:
  sum = sum + a
  a = a + 1
print(sum)
```
```
使用while 迴圈(loop)計算1+3+5+7.....+99
```
```
a = 1
sum = 0
while a < 100:
  sum = sum + a
  a = a + 2
print(sum)
```
```
使用while 迴圈(loop計算1*3*5*7.....*99
```
```
a = 1
sum = 1
while a < 100:
  sum = sum * a
  a = a + 2

```
